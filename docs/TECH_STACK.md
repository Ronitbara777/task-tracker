# Tech Stack

## Frontend
- **React**: It's component-based, predictable, and makes handling UI state straightforward.
- **Vite**: Provides an incredibly fast development server and optimized build process compared to older bundlers like Webpack.
- **Tailwind CSS**: Allows for rapid UI styling directly in the markup without having to jump between CSS files, keeping the design system consistent.

## Backend
- **Node.js**: Lets us write the backend in JavaScript, keeping context switching to a minimum.
- **Express**: A minimal and flexible web framework that makes setting up routing and middleware quick and painless.

## Database
- **MongoDB**: A NoSQL database that maps perfectly to JSON objects, making it very natural to work with in a JavaScript stack.
- **Mongoose**: Provides schema validation and easy querying on top of MongoDB, catching bad data before it hits the database.

## Deployment
- **Vercel (Frontend)**: Perfect for static sites and SPAs, handles routing automatically with minimal config.
- **Render / Heroku (Backend)**: Straightforward platforms for deploying Node servers with zero fuss.
